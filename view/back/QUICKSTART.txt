📊 STRUCTURE TABLE PRODUIT
───────────────────────────────────

id                 → INT PRIMARY KEY AUTO_INCREMENT
nom                → VARCHAR(255) NOT NULL
description        → TEXT
prix               → DECIMAL(10,2) NOT NULL
quantiteStock      → INT DEFAULT 0
dateExpiration     → DATE NOT NULL
estDurable         → TINYINT(1) DEFAULT 0
image              → VARCHAR(255)
statut             → VARCHAR(50) AUTO CALCULÉ
id_categorie       → INT DEFAULT 1
created_at         → TIMESTAMP AUTO
updated_at         → TIMESTAMP AUTO

═══════════════════════════════════════════════════════════

🤖 STATUT AUTO-CALCULÉ
───────────────────────────────────

La fonction determinerStatut() recalcule le statut automatiquement :

✅ DISPONIBLE  (vert)    → stock > 0 ET date ok
  ⚠️  RUPTURE    (orange)  → stock = 0 ET date ok
❌ ÉPUISÉ      (rouge)   → date dépassée

❓ DÉPANNAGE
───────────────────────────────────

Q: Erreur "Table 'produit' doesn't exist"
A: Exécuter schema.sql via PhpMyAdmin

Q: Images non affichées
A: Vérifier dossier uploads/ existe et permissions 755

Q: Erreur upload "Permission denied"
A: chmod 755 uploads/

Q: Statut ne s'update pas
A: Normal ! Statut recalculé à chaque affichage
